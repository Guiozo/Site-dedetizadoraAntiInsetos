# 🐜 AntiInsetos - Site de Dedetizadora

Site institucional desenvolvido para a empresa **AntiInsetos Dedetizadora**, com foco em apresentação de serviços e captação de clientes via WhatsApp.

---

## 🚀 Demonstração

Este projeto é um site estático, leve e responsivo, ideal para pequenas empresas que desejam presença online rápida e eficiente.

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3 (com variáveis e responsividade)  
- JavaScript (interação com formulário)  
- Google Fonts (Poppins)  
- Font Awesome (ícones)

---

## 📌 Funcionalidades

- ✅ Layout moderno e responsivo  
- ✅ Seção de apresentação (Hero)  
- ✅ Listagem de serviços oferecidos  
- ✅ Formulário de orçamento integrado ao WhatsApp  
- ✅ Botão flutuante de contato rápido  
- ✅ Navegação suave entre seções  

---

## 📱 Integração com WhatsApp

O site permite que o cliente envie uma mensagem diretamente para o WhatsApp da empresa com:

- Nome  
- Tipo de serviço  
- Detalhes adicionais  

Tudo isso automaticamente formatado ao enviar o formulário.
